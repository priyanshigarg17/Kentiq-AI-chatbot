## Project Structure

```
kentiq_project/
│
├── .env                    ← Environment variables (API keys, settings)
├── config.py               ← Reads .env, exposes Config class
├── requirements.txt        ← All Python dependencies
│
├── generate_pdf_data.py    ← Run ONCE to create the bank data PDF
├── pdf_reader.py           ← Reads/parses bank data from PDF
│
├── intent_detector.py      ← NLP-lite: speech text → intent
├── tts_module.py           ← Text-to-Speech (gTTS / pyttsx3 / Azure)
├── stt_module.py           ← Speech-to-Text (Whisper / Google / Sphinx)
│
├── balance_module.py       ← Account balance inquiry (Milestone 2)
├── transfer_module.py      ← Money transfer workflow (Milestone 3)
├── cheque_module.py        ← Cheque image validation (Milestone 4)
├── kyc_module.py           ← Voice/Video KYC recording (Milestone 5)
├── error_handler.py        ← Error handling & logging (Milestone 6)
│
├── conversation_manager.py ← Master controller: wires all modules
├── main.py                 ← FastAPI backend server (all REST endpoints)
│
├── data/
│   └── bank_dummy_data.pdf ← Generated by generate_pdf_data.py
├── uploads/                ← Validated cheque images saved here
└── kyc_recordings/         ← KYC audio/video files saved here
```

## Setup Instructions

### Step 1 — Clone / Download the project
```bash
cd kentiq_project
```

### Step 2 — Create virtual environment
```bash
python -m venv venv
source venv/bin/activate        # Linux / macOS
venv\Scripts\activate           # Windows
```

### Step 3 — Install dependencies
```bash
pip install -r requirements.txt
```

### Step 4 — Configure environment
```bash
cp .env .env.backup             # Backup default
# Edit .env and fill in your API keys:
#   OPENAI_API_KEY=your_key_here    (for Whisper STT)
#   TTS_ENGINE=gtts                 (or pyttsx3 for offline)
#   STT_ENGINE=google               (or whisper)
```

### Step 5 — Generate the bank data PDF (run ONCE)
```bash
python generate_pdf_data.py
# Creates: data/bank_dummy_data.pdf
```

### Step 6 — Start the FastAPI server
```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### Step 7 — Open API docs
```
http://localhost:8000/docs
```

## API Endpoints

| Method | Endpoint               | What it does                      |
|--------|------------------------|-----------------------------------|
| GET    | /api/health            | Server health check               |
| POST   | /api/welcome           | Mandatory welcome message + audio |
| POST   | /api/chat              | Text message → bot response       |
| POST   | /api/transcribe        | Audio file → transcript text      |
| POST   | /api/speak             | Text → TTS audio file             |
| GET    | /api/audio/{filename}  | Serve generated audio file        |
| GET    | /api/balance           | Get account balance from PDF      |
| GET    | /api/transactions      | Get recent transactions from PDF  |
| POST   | /api/cheque            | Upload + validate cheque image    |
| POST   | /api/kyc               | Start KYC recording session       |
| GET    | /api/kyc/recordings    | List all saved KYC recordings     |
| POST   | /api/session/reset     | Reset a conversation session      |

## TTS Engine Options

| Engine   | Quality | Internet | Cost  | Setup                          |
|----------|---------|----------|-------|--------------------------------|
| `gtts`   | Good    | Required | Free  | `pip install gtts`             |
| `pyttsx3`| Fair    | No       | Free  | `pip install pyttsx3`          |
| `azure`  | Best    | Required | Paid  | Azure key + SDK                |

Set in `.env`: `TTS_ENGINE=gtts`
